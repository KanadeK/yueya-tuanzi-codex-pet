# 妙脆角小猫 1.0.0

The first public release provides a Codex Desktop pet built from the owner-approved reference image without AI redraw.

## Included

- 9 native Codex status animations derived from one source-faithful cutout
- 1536 × 1872 transparent lossless WebP atlas
- `spriteVersionNumber: 2` metadata
- Windows, macOS, and Linux installers with SHA-256 verification
- Deterministic cutout builder, asset builder, validator, tests, and preview site
- Source and license-boundary documentation
